# Flet Tax Calculator

PythonのGUIフレームワーク「Flet」を使用して作成した、実用的な電卓アプリケーションです。

## 特徴
- **税込計算機能**: 日本の消費税（10%）をワンタップで計算できる専用ボタンを実装。
- **レスポンシブな操作感**: ボタンのクリックイベントに応じた動的な表示更新を実現。
- **シンプルな内部ロジック**: Pythonの `eval()` 関数を活用し、文字列としての数式を評価・計算。

## 使用技術
- Python 3.x
- Flet (GUI Framework)

## 開発の背景
GUIアプリ開発の基礎である「レイアウト設計（Row/Column）」と「イベント処理」を習得するために制作しました。日常的に使う電卓に、自分にとって便利な「税込ボタン」を追加することで、実用性を高める工夫をしました。
